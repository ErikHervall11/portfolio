Cocktail Collective

Cocktail Collective is a dynamic and interactive website designed for cocktail enthusiasts. Built using Flask and Python for the backend, and React Redux for the frontend, this platform allows users to explore and share cocktail recipes.

BeatRate

BeatRate is an interactive platform designed for music enthusiasts to discover, rate, and review albums. Developed with Flask and Python for the backend and React Redux for the frontend, BeatRate offers a user-friendly interface where users can create profiles, explore a vast database of albums, and share their opinions.

erabnb

Erabnb is a unique vacation rental platform that allows users to book homes in various historical time periods. Built with Express and Node.js for the backend and React Redux for the frontend, the site offers a distinctive twist on traditional vacation rentals.
